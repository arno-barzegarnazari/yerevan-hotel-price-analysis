This analysis was conducted as a Final Project for Business Analytics for Data Science course in American University of Armenia.

The project analyzes hotel pricing in Yerevan using statistical and machine learning techniques to identify the key factords influencing daily room rates. By examining features such as star ratings, customer satisfaction scores, and available amenities, the analysis provides data-driven insights that support more effective pricing strategies and competitive positioning in the local hospitality market.

To successfully run the code, please install the required dependencies. You can find those libraries in the import section of the code notebook.
